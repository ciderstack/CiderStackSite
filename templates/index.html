Signature Feature

Instant snapshots for MDM testing

Capture VM state before enrollment. Test your MDM profile. Restore in under a second. Powered by APFS copy-on-write—no waiting, no duplicate disk space.

Test MDM configs without the reset dance

MDM testing usually means wiping and re-imaging after every failed enrollment. With CiderStack snapshots, you're back to a clean state instantly.

✓
Sub-second restore APFS clonefile means instant copy-on-write snapshots
✓
No extra disk space Snapshots only store the delta, not full copies
✓
GUI or CLI Right-click → Snapshots, or script it in your pipeline
Terminal 
# Snapshot before MDM enrollment
cider snapshot create myvm "Before MDM"
→ Created snapshot abc123

# Test your MDM profile...
# Something went wrong? Restore instantly:
cider snapshot restore myvm abc123
→ Restored in 0.3s

# List all snapshots
cider snapshot list myvm
abc123  "Before MDM"     2m ago
def456  "Post-install"   1h ago
Signature Feature

VMs that know when to retire

Set a time-to-live when you create a VM. When it's done, CiderStack retires it automatically. No more orphaned test VMs hogging your storage.

Intent-based lifecycle management

Tag each VM with its purpose—Interactive, Disposable, CI, or Persistent—and CiderStack handles the rest. Disposable VMs get auto-cleaned; persistent ones stick around.

Features

Everything else you'd expect

Native Performance

Built on Virtualization.framework—Apple's hypervisor. No emulation. Near-native speed on Apple Silicon.

SwiftUI Dashboard

Card-based VM overview, real-time resource monitoring, intuitive creation wizard. A VM manager that looks like a Mac app.

OCI Image Support

Push, pull, and share VM images like Docker containers. Use any OCI-compatible registry.

Premium CLI

Clean, predictable commands with --json output for scripting. Designed for automation and CI/CD.

Apple Silicon First

Optimized for M1/M2/M3/M4. Intel Macs supported, but the real magic happens on Apple Silicon.

IPSW Import

Drag and drop Apple IPSW files to create base images. Supports Sonoma, Sequoia, and Tahoe.
